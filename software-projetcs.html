1. The sensor-logger

The Sensor Logger project combines both hardware and software components to efficiently gather sensor data on-site. This collected data is securely stored in a database and subsequently uploaded to a website platform. The platform offers user account creation, allowing individuals to access and analyze the data conveniently through tables.
