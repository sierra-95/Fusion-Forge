The Sensor Logger

The Sensor Logger project is an IoT project that eliminates the need of enviromental data analysts from collecting data physically in this time of evolving technology.

Sensors placed according to the owners desire efficiently gather data on-site.This collected data is continuously stored in a database and can be accessed through a web application.

Users can view timestamped data as raw values or graphical representations and can also export it for further analysis.

Tomorrow, your personal task manager

Tomorrow is a web application designed to assist users get the best of task priority, productivity analysis and goal tracking.

It has its own dynamic calender and a dedicated mail server which serves as your personal secretary, reminding you of every event.

It also features a self development section which has a FutureMe section, a productivity analysis section and recommended speeches and books one can read.
